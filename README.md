# Django编写的简易个人博客

## 环境要求

- python 3.9.0
- Pycharm Community 2021.1.3
- mysql 8.0.26

- Django 4.0.1
- Markdown 3.3.6
- Pillow 9.0.1
- Pygments 2.11.2
- django-bootstrap4 21.2
- django-imagekit 4.1.0
- mysqlclient 2.1.0

## 程序结构

- Blog

  Blog为程序项目

- blogs

  blogs为应用程序，包含文章等信息

- comments

  comments为应用程序，包含评论等信息

- users

  users为应用程序，包含用户等信息

## 目前已实现的功能

1. 文章发表、文章修改、文章查看
2. 发表评论、修改评论
3. 用户注册、用户登录、用户退出、修改密码
4. 用户个人主页、编辑个人介绍、上传个人头像
5. markdown渲染博客（代码高亮貌似有点问题- -）
6. 分页展示
